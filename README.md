# 💊 영양제 알람 (NutriAlarm)

> 코딩 경험이 전혀 없는 사용자가 AI(Claude)와 **대화만으로** 만든 영양제 관리 앱입니다.
> 서버 없이 완전 무료로 운영됩니다.

**👉 바로 사용하기: https://mine-0324.github.io/nutrialarm/**

스마트폰 브라우저로 열고 "홈 화면에 추가"하면 일반 앱처럼 사용할 수 있습니다 (아이폰 Safari / 갤럭시 삼성인터넷·크롬).

## 주요 기능

| 기능 | 설명 |
|---|---|
| ⏰ 복용 알림 | 버튼 하나로 휴대폰 캘린더에 매일 반복 알람 등록 (.ics) |
| ✅ 복용 체크 | 시간대별 체크리스트, 오늘 복용률 표시 |
| 📅 복용 달력 | 날짜별 복용 기록을 색깔로 표시 |
| 🔍 제품 검색 | 식약처 등록 건강기능식품 **49,000여 개** 이름 검색 → 자동 입력 |
| 📷 라벨 사진 인식 | 성분표를 찍으면 성분·함량 자동 입력 (휴대폰 내 처리, OCR) |
| 🧮 함량 분석 | 하루 총 섭취량을 **나이·성별 맞춤 기준**(2020 한국인 영양소 섭취기준)과 비교, 상한 초과 경고 |
| 🔗 조합 분석 | 칼슘+철분 같은 흡수 방해 조합 경고, 좋은 조합 안내 |
| 💡 복용 시간 추천 | 성분별 권장 시간 안내 (철분=아침 공복, 마그네슘=저녁 등) |
| 💾 백업 | 기록을 파일로 내보내기/불러오기 |

## 기술 구성

- **HTML/CSS/JavaScript 단일 파일 웹앱(PWA)** — 프레임워크·빌드 도구 없음
- **호스팅**: GitHub Pages (무료)
- **데이터 저장**: 브라우저 localStorage (기기 내 저장, 서버 전송 없음)
- **글자 인식**: Tesseract.js (기기 내 처리)
- **제품 데이터**: 식품의약품안전처 공공데이터
  - 건강기능식품 품목제조신고 현황 (식품안전나라 OpenAPI, I0030)
  - 전국 건강기능식품 영양성분정보 표준데이터 (공공데이터포털)
